# AI Feature Prioritization Engine

## Problem Statement

Product managers receive thousands of customer reviews and feature requests. It is difficult to manually identify major product issues and decide which features should be prioritized first.

This project builds an AI-powered decision engine that automatically analyzes user reviews, detects product pain points, clusters similar complaints, and prioritizes feature development using the RICE product management framework.

## Tech Stack

- Python
- Google Colab
- Hugging Face Transformers
- Sentence Transformers
- Scikit-learn
- Plotly
- KMeans Clustering

## Workflow

Customer Reviews  
↓  
Sentiment Analysis  
↓  
Embedding Generation  
↓  
Complaint Clustering  
↓  
Feature Suggestion  
↓  
RICE Scoring  
↓  
Dashboard  

## Output

AI-generated prioritized product roadmap
